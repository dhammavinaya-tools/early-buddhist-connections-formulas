# SN45.133-138 The Ocean

(In this version §§127–132 are identical with §§91–96, and §§133–138 with §§97–102, except for the following change:)

“Here, bhikkhus, a bhikkhu develops and cultivates right view … right samādhi, which slants, slopes, and inclines towards Nibbāna.”
